# Smart Automation Suite for Insurance 🚀

Collection of n8n workflows including OTP authentication, insurance policy verification, and AI-based damage analysis.

---

## 🔧 Workflows Included

- 🔐 OTP Authentication Workflow  
- ✅ OTP Verification Workflow  
- 🛡️ Insurance Policy Verification  
- 🚗 AI Damage Analysis Workflow  

---

## ⚙️ Setup Instructions

1. Import the JSON files into n8n
2. Configure environment variables:
   - SUPABASE_URL
   - API_KEYS
   - TWILIO_CREDENTIALS
3. Activate the workflows

---

## 📌 Use Cases

- User authentication via OTP  
- Insurance policy validation automation  
- AI-based car damage detection  

---

## ⚠️ Important

Sensitive data (API keys, URLs, credentials) has been removed for security reasons.
